# Build Core Router

Install and configure firewall/router.

---

## Current Status

No automated install exists for Core Router.

---

## Manual Install Procedure

1. Create bootable USB with OPNsense/pfSense image
2. Boot from USB and complete installer
3. Apply base configuration via `deevnet.net` Ansible collection
