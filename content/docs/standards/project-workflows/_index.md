# Project Workflow Templates

Reusable milestone structures for bootstrapping new projects with consistent organization.

## Purpose

These templates provide proven workflow patterns extracted from successful projects. When starting a new project, copy the appropriate template to your roadmap and adapt the tasks to your specific needs.

## Available Templates

- **[Embedded Hardware](embedded-hardware/)** - Hardware projects with firmware, PCB design, and physical enclosures. Follows EVT/DVT/PVT lifecycle.
- **[Infrastructure Automation](infrastructure-automation/)** - Infrastructure as Code projects for provisioning and managing systems.
- **[Application Development](application-development/)** - Software projects including services, tools, and applications.

## Usage

1. Select the template that best matches your project type
2. Copy the milestone structure to your project's roadmap page
3. Customize task lists for your specific requirements
4. Remove milestones that don't apply
5. Add project-specific milestones as needed

Templates are starting points, not rigid constraints.
